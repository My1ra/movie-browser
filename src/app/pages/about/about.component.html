About Movie Explorer

Movie Explorer is a simple yet powerful web application that lets you discover popular movies and view detailed information about them. All movie data is fetched from the TMDb (The Movie Database) API, ensuring you get up-to-date and accurate information.

This project is built using Angular, leveraging the power of Angular Material for responsive and modern UI components. The app demonstrates key Angular concepts such as standalone components, lazy loading, routing, and state management.

The goal of this application is to provide a clean, intuitive interface to explore movies and learn modern Angular development practices. It is designed to be scalable and maintainable, allowing for easy feature extension and improvements.
